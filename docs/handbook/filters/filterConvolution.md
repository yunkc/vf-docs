# FilterConvolution

## 调整滤镜
滤镜地址:
> https://s.vipkidstatic.com/vf/plugin/FilterConvolution/0.0.1.js

滤镜名
> ConvolutionFilter 

## 属性

| 属性名 | 属性类型 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| width | number | 0 | 滤镜单位宽度 |
| height | number | 0 | 滤镜单位高度 |
| matrix | array\<number> | 0 | 滤镜矩阵 |
